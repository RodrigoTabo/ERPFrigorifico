# ERPFrigorifico
**ERPFrigorifico es un sistema integral para la industria frigorifica. Desarrollado en ASP.NETCore Web Api, diseñado para optimizar control de materia prima por proveedores, stock logistico y ventas por pedidos. El proyecto incorporara metricas para apoyar la toma de decisiones estrategicas mediante eventos y notificaciones.**

## **Funcionalidades y eventos**
- **Gestion de Produccion: Control de produccion y seguimiento del proceso completo.**
- **Control de Materia Prima: Registro y seguimiento de insumos para asegurar disponibilidad y minimizar desperdicios.**
- **Stock Logistico: Gestion de inventarios, entradas, salidas y movimientos internos.**
- **Ventas y Pedidos: Manejo completo de pedidos, clientes y facturacion.**
- **Metricas y KPIs: Indicadores clave para decisiones rapidas y efectivas.**

# **Tecnologias**
- **Backend: ASP.NET Core Web API**
- **Base de Datos: SQL Server con Entity Framework Core**
- **Autenticacion y Roles: IdentityCore (Roles integrados sin heredar IdentityUser para evitar sobre-ingenieria.)**
- **Arquitectura: Separacion por capas para mantenibilidad y escalabilidad**

## **Objetivo**
La idea es brindar un sistema robusto, escalable y eficiente que permita a empresas del sector frigorifico gestionar operaciones complejas y obtener informacion valiosa para la toma de decisiones estrategicas.
